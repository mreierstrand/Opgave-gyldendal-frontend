
#  Gyldendal-opgave
Frontend opgave lavet til Gyldendal

Brug npm til at køre projektet
-

> npm run dev

Brug Cypress til test af projektet
-

1. Installér Cypress

  

> npm install cypress --save-dev

  

  

2. Kør Cypress test

  

> npx cypress open
(mens "npm run dev" kører)

  
  

3. Vælg E2E Testing

  
  

4. Vælg Firefox

  
  

5. Vælg myTest.cy.js

  
  

6. Se om testen passer eller failer
